About EasyCart

EasyCart is a beginner-friendly mini e-commerce website created using HTML, CSS and JavaScript. The main purpose of this project is to understand how online shopping websites work.

🛒 Our Mission

Our mission is to provide snacks, drinks and instant food items in a simple, fast and user-friendly way.

💻 Technologies Used

This website is built using:

• HTML for structure
• CSS for styling
• JavaScript for functionality

⚡ Features

• Product Search
• Category Filter
• Add to Cart
• Quantity Control
• Dark Mode
• Responsive Design

🎯 Project Goal

The goal of EasyCart is to learn real-world JavaScript concepts in a practical and interactive way.
